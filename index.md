## About Git Visor

Git Visor is a web application and Electron app built with React and TypeScript. It provides an intuitive interface to visualize the internal state of a Git repository, making it easier to understand and manage your version control workflow.

<box type="info">

Git Visor is designed for developers who want to gain deeper insights into their Git repositories. Whether you're a beginner or an advanced user, Git Visor offers tools to help you navigate and manage your Git history effectively.

</box>
