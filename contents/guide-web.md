# How to Use
1. Visit the web app at [Git Visor Web App](https://git-visor.github.io/web-app/)
2. You can select any sample repository with the "Select Repository" dropdown menu at the top left. Each sample repository shows a different aspect of Git. For example, the "File Deletion Example" repository demonstrates how file deletions are represented in Git.
3. You can also select the branch to visualise using the "Select Branch" dropdown menu next to it. Currently, only the "branch-sample" repository has multiple branches to choose from.
4. <a id="load-url"></a>[Optional] If you want to visualise your own Git repository, you can paste the json link generated from the [desktop app's "Export JSON"]({{baseUrl}}/contents/guide-desktop.html#export-json) into the "Load URL" input box and click "Load".
